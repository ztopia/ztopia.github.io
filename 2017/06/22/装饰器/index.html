#装饰器
装饰器本质上就是一个函数

##类装饰器

@decoration

class Person {
static arr:any[]=[1,2,3]
}

Person类就是需要装饰的原型，@decoration就是调用哪个函数来装饰这个类

function decoration(target:any):void{
let arrNew=[4,5,6];
if(target.arr){
target.arr.push(…arrNew)
}
}

要实现装饰功能，需要将之前的类传递给装饰函数，这个函数也是用来服用的函数
target用来接收来自类的属性和方法

##方法装饰器

class Person {
@readonly
run(): void {
console.log(‘跑’)
}
}

//这里的run()是一个实例对象，所以，装饰器接收到的原型

function readonly(…arg: any[]) {
arg[2].writable = false;
}

// 自动收到的参数1, 实例方法为原型, 静态方法为类
// 自动收到的参数2, 为方法的名称
// 自动收到的参数3, 为方法的描述对象
这里将第三个参数中的writable属性做了更改。

##属性装饰器

class Person {
@propZhuangshiqi
name: string = ‘匿名’;

@propZhuangshiqi
static age: number = 0;

}

属性分为实例属性和静态属性

function propZhuangshiqi(…arg: any[]): void {
let classOrPrototype = arg[0]; // 类或原型
let propName = arg[1]; // 属性名称

// 根据属性名称添加对应的set方法
classOrPrototype['set' + propName] = function(val: any): void {
    this[propName] = val;
}

// 根据属性名称添加对应的get方法
classOrPrototype['get' + propName] = function(): any {
    return this[propName];
}

}

装饰!!实例!!属性接收到的第一个参数为!!原型!!, 第二个参数为要装饰的属性名称
装饰!!静态!!属性接收到的第一个参数为!!类!!, 第二个参数为要装饰的属性名称

Person.prototype.setname(100)
Person.prototype.getname() //100

静态方法直接调，静态方法需要prototype
实例方法可以直接调用

##装饰器传参

@send(135)
class Person{}

在装饰器后面添加需要传入的参数

function send(a:number){
return function(target:any){
console.log(a); //135
console.log(target) //Person
}

}

装饰函数的外层接收这个参数，内层接收这个类，这样就可以在装饰函数内部使用这个参数了

##举例
方法一：
// 装饰器工厂, 专门用来装饰方法
function getDecorator() {
return function(…arg: any[]) {

    let oldFn = arg[2].value;  // 缓存旧方法
    console.log(oldFn);

    // 装饰方法自动接收的第三个参数为描述对象,该对象有个value属性,可以获取被装饰的方法自身
    // 把旧的方法替换成新的方法, 新的方法会记录执行时间
    arg[2].value = function(count: number) {
        let startTime = Date.now();     // 记录方法执行的开始时间
        oldFn(count);
        console.log(Date.now() - startTime);   // 打印方法执行所需的时间
    };
}

}

方法二：
// 时间统计的装饰器
function time(target: any, name: string, msg: any) {
// 保存旧方法
let oldFn = msg.value; 

// 替换成新方法
msg.value = function() {
    let startTime = Date.now();
    oldFn();
    console.log(Date.now() - startTime);
}

}

@time           //调用工厂函数需要加（）

    }

}

Dog.loop();
